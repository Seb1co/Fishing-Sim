# Fishing Sim
A simple fishing simulator where you grind for better gear by fishing.Random events will spawn every 1-5 minutes,these events give you better chances at better fish .
# Controls
I -> Show the whole inventory </br>
E -> opens the shop when in range </br>
Left_Clik -> fishing,purchasing items when the cursor is hovering the buttons,selecting the type of gear that will be shown in the shop </br>
W,A,S,D -> for the movement </br>
